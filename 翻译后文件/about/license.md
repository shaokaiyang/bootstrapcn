- 你必须在你使用的 Bootstrap 的 CSS 和 JavaScript 文件中保留许可和版权声明。

- 免费下载和使用整体或部分 Bootstrap ，无论用于个人，私人，公司内部或者商业用途

- 持有作者和许可证拥有者的赔偿责任，因为以 Bootstrap 无担保形式提供

- 包括 Bootstrap 自身在内，或您可能对它做的任何修改，你可以组装任何重新发布的版本
